# Hospital Management System

## Technologies Used
- Java
- SQL (MySQL)


## Description
A console-based Hospital Management System developed using Core Java and SQL.
The system helps manage patients, doctors, and appointment scheduling.

## Features
- Add and view patient details
- Add and view doctor details
- Book appointments
- View appointment schedules
- Store data using SQL database

## Database
- MySQL
- Tables: Patient, Doctor, Appointment

## How to Run
1. Import the SQL file and create database tables
2. Update database credentials in Java file
3. Compile and run the main Java class
